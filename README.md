# Routeplane Provider Registry

Community-contributed provider configurations for Routeplane.

## Structure

```
dist/
  <provider-id>.toml
  <provider-id>.toml
```

Each `.toml` file defines a provider's models, auth, and endpoints.

## Contributing

1. Fork this repo
2. Add your provider TOML under `dist/`
3. Open a PR
